# AGENTS.md

## Purpose
This is the repo entry point for humans and agents. Read this file first; the
standards files under `docs/standards/` are the authoritative source for all
rules. This file is an index, not a duplicate.

## Required Read Order
1. `AGENTS.md`
2. `docs/README.md`
3. `docs/repo-map.md`
4. the task's design doc under `docs/plans/architect/`
5. task-relevant standards in `docs/standards/`
6. task-relevant audits or spelunks under `docs/spelunk/`
